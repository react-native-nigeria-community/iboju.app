## About

Iboju.app (formerly **PreviewKit**) is a simple web tool that helps you generate clean, professional app preview images. Upload your screenshots, place them inside device frames, customize background colors or gradients, add text, and export highâ€‘quality marketing images.

## Features

* Upload and manage app screenshots
* Insert screenshots into iOS/Android device templates
* Edit frame text and layout
* Choose solid colors or gradients for backgrounds
* Export final preview as a single image
* Built with **React + Vite** for speed and simplicity

## Tech Stack

* **React**
* **Vite**
* **TailwindCSS**
* **TypeScript** (optional depending on project setup)
* **Redis**
